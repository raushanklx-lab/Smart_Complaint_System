# Smart Complaint Management & Tracking System

## 📌 Project Overview

The Smart Complaint Management & Tracking System is a web-based application developed to simplify the process of registering, tracking, and managing complaints. It allows users to submit complaints online, monitor their status, and enables administrators to efficiently manage and resolve complaints through a dedicated admin dashboard.

---

## 🚀 Features

### User
- User Registration & Login
- Submit Complaint
- Upload Complaint Image
- View Complaint History
- Track Complaint Status
- Edit Profile
- Change Password
- Logout

### Admin
- Secure Admin Login
- View All Complaints
- Search Complaints
- Update Complaint Status
- Delete Complaints
- Export Complaints to CSV
- Dashboard Statistics

## ⚙️ Installation

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create MySQL database

Import the `database.sql` file into MySQL.

4. Configure `config.py`

Update your MySQL credentials.

5. Run the application

```bash
python app.py
```
